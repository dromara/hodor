#### 简介
hodor ui 是基于 [pear-admin-ant](https://gitee.com/pear-admin/pear-admin-ant) 二次开发。

#### 快速上手

```
// 依赖安装
npm install

// 开发预览
npm run serve

// 开发打包
npm run build

// 生产打包
npm run build-prod

// 项目分析
npm run analyzer
```
